9. Import/Export
Menü ->File -> Import
Menu -> File -> Export


Die Im- und Exporter dienen dazu, Dateiformate von anderen Programmen/Spielen/... zu laden bzw. in dieses Format zu speichern. Beim Importieren wird die neu geladene Szene immer mit der aktuellen Szene verschmolzen, beim Exportieren wird die ganze Szene gespeichert.

9.1 Import

9.1.1 MD2: Quake 2
Menü ->File -> Importer -> Quake II MD2


MD2 ist das Dateiformat, das im Spiel 'Quake II' verwendet wird. Es speichert Animationen, welche allerdings hier nicht geladen werden. Nachdem das Model importiert wurde, befindet sich nur ein Material ohne Textur darauf. Um dem Model seine Textur zu geben (Skin), muss diese im Material-Editor ausgewählt werden.

9.1.2 3ds: 3d Studio
Menü ->File -> Import -> Autodesk 3DS


3DS ist das Autodesk 3D Studio Model Format. Das Modelformat kann zusätzlich zu den statischen Daten noch zusätzliche Animationen abspeichern, welche aber beim Importieren ignoriert werden.
Einstellungen/Parameter
Filename: Der Dateipfad des zu ladenden Modells
Texture directory: Der Pfad zu den Texturen. Wird das Feld leer gelassen, werden die Texturen aus dem gleichen Ordner wie die zu ladende Datei verwendet
Disable loading textures: Erzwingt, dass keine Texturen geladen werden
Center mesh: Gibt an, ob das geladene Modell zentriert in die Szene eingefügt werden soll
Scale to max. size: Gibt an, ob das geladene Modell auf eine bestimmte maximale Größe skaliert werden soll
Set specular power 0.0 to specific value: Gibt an, ob ein Glanzfaktor von 0.0 mit einem anderen Wert ersetzt werden soll
Negate v-coordinate of texture: Gibt an, ob die v-Texturecoordinate negiert werden soll

9.1.3 OBJ: Lightwave
Menü ->File -> Import -> Lightwave OBJ


OBJ ist das Modelformat von Lightwave. Es werden die Vertex-, Material- und Gruppendaten gelesen, alle anderen Daten der Datei werden ignoriert.
Einstellungen/Parameter
Filename: Der Dateipfad des zu ladenden Modells
Texture directory: Der Pfad zu den Materialfiles (mtl). Wird das Feld leer gelassen, werden die Materialien aus dem gleichen Ordner wie die zu ladende Datei verwendet
Texture directory: Der Pfad zu den Texturen. Wird das Feld leer gelassen, werden die Texturen aus dem gleichen Ordner wie die zu ladende Datei verwendet
Disable loading textures: Erzwingt, dass keine Texturen geladen werden
Center mesh: Gibt an, ob das geladene Modell zentriert in die Szene eingefügt werden soll
Scale to max. size: Gibt an, ob das geladene Modell auf eine bestimmte maximale Größe skaliert werden soll
Set specular power 0.0 to specific value: Gibt an, ob ein Glanzfaktor von 0.0 mit einem anderen Wert ersetzt werden soll

9.2 Export

9.2.1 OBJ: Lightwave
Menü ->File -> Export -> Lightwave OBJ


OBJ ist das Modelformat von Lightwave. Es werden die Vertex-, Material- und Gruppendaten gespeichert, alle anderen Szenendaten (z.b. Szenenkommentar) werden ignoriert. Die generierte Materialfile (mtl) befindet sich im gleichen Ordner wie die OBJ-Datei und hat den gleichen Namen wie diese Datei.
