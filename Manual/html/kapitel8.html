8. Serialisierung

8.1 Neue Szene erstellen
Menü -> File -> New

Es wird eine neue Szene angelegt, alle bisher ungespeicherten Änderungen gehen verloren. Auch die erstellten Materialien sind davon betroffen.

8.2 Szene speichern
Menü -> File -> Save
Menü -> File -> Save as

Mit Save wird die aktuelle Szene gespeichert. Falls für die aktuelle Szene noch keine Dateiname vergeben wurde, wird nach einem Dateinamen gefragt. Auch das Speicherverzeichnis kann bei dieser Gelegenheit gewählt werden.

Mit Save as wird immer nach einem neuen Dateinamen nachgefragt.

8.3 Szene laden
Menü -> File -> Load

Vorhandene gespeicherte Szenerien können mit Load geladen werden. Alle nicht gespeicherten Änderungen der aktuellen Szene gehen dabei verloren.

8.4 Szene importieren
Menü -> File -> Merge

Eine abgespeicherte Szene kann zur aktuell geladenen Szene hinzugefügt werden.

Dabei wird explizit nachgefragt, wessen Materialbibliothek verwendet werden soll, da eine Mischung beider Materialbibliotheken nicht möglich ist.

8.5 Szene kommentieren
Menü -> File -> Comment scene

Hier kann zu einer Szene ein beliebig langer Kommentar hinterlegt werden, der auch mit der Szene mitgespeichert wird. Dies kann etwa dazu verwendet werden um die nächste ausstehende Modellierungstätigkeit anzugeben.
