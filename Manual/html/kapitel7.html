7. Materialien
Um Oberflächen zu beschreiben, bietet BlackSun Materialien an. Diese kombinieren Oberflächeneigenschaften (z.b. Farben), alle zu verwendenden Texturen und die Kombinationseinstellungen dieser Texturen. Hat ein Modell kein Material gebunden, wird ein Standardmaterial verwendet.

7.1 der Materialeditor
Menü -> Material -> Material editor


Der Materialeditor dient dazu, alle Oberflächeneigenschaften komfortable zu konfigurieren.
Material-Editor
Der Editor ist in 5 Teile geteilt:
Available materials: Eine Liste mit allen verfügbaren Materialien
Preview: Ein Vorschaufenster
General: Farbeinstellungen und Name des Materials
Texturestate: Liste der auf das Material gelegten Texturen und deren Einstellungen
Buttonfeld: Verwalten von Materialien

Available materials:
Alle verfügbaren, vom Benutzer erstellten Materialien werden hier aufgelistet. Es wird der Name des Materials und die ID aufgezeigt.

Preview:
Zeigt eine Vorschau des Materials an bestimmten Grundkörpern dar. Einstellen lässt sich die Vorschau über das Kontextmenü (erreichbar über die rechte Maustaste):




Preview-Kontextmenü
Reset view: Setzt die Kameraeinstellungen zurück
Show grid: Aktiviert/Deaktiviert das Gitternetz
Render box: Stellt als Testobjekt eine Box dar
Render sphere: Stellt als Testobjekt eine Kugel dar
Render torus: Stellt als Testobjekt einen Torus dar
Render helix: Stellt als Testobjekt eine Spirale dar
Render grid: Stellt als Testobjekt ein Gitter dar
Render geosphere: Stellt als Testobjekt eine GeoSphere dar

General:
Hier können der Name und die Farbeigenschaften eingestellt werden. Es können folgende Farbeigenschaften bearbeitet werden:

Color: Die Grundfarbe
Ambient color: Ambienter Lichtanteil der reflektiert werden soll
Diffuse color: Diffuser Lichtanteil der reflektiert werden soll
Specular color: Glanzlichtanteil der reflektiert werden soll
Emissive color: Selbstaustrahlender Anteil
Specular factor: Der Glanzfaktor (Glanzlichtexponent). Je größer der Faktor, desto härter/glätter wirkt das Material

Texturestate:
Texturen können hier geladen und auf das Material gelegt werden. Insgesamt können maximal 4 Texturen kombiniert werden. Die Art und Weise, wie diese miteinander kombiniert werden sollen, können über die weiterführenden Eigenschaften bestimmt werden. Um eine Textur zu laden, klicken sie auf die ComboBox nebem der Texturestate und wählen “Load from file ...” aus. Sind Texturloaderoutinen vorhanden, können nun Texturen ausgewählt und geladen werden. Standardmäßig mit dem mitgeliefertem Texturladeroutinen lassen sich BMP-, GIF-, JPEG-, JPG-, PBM-, PGM-, PNG-, PPM-, TGA-, XBM- und XPM-Dateien laden. Nach dem Laden befindet sich die neue Textur in der Liste und kann ausgewählt werden. Um eine Textur von einem Texturestate zu entfernen, wählen Sie “-No-” aus.

Um die weiterführenden Eigenschaften aufzuzeigen, klicken Sie auf das Kreuz nebem der Texturestate. Folgende Werte können eingestellt werden:

Color Scaling: Der Faktor, mit dem der Farbwert multipliziert werden soll. Zur Verfügung stehen die Multiplikatoren 1, 2 und 4
Combine RGB: Stellt die Art ein, wie die RGB-Farbwerte miteinander kombiniert werden sollen
Combine Alpha: Stellt die Art ein, wie der Alpha-Farbwert miteinander kombiniert werden soll
ColorArg0-2: Der Wert für die Operatoren 0, 1 und 2 für die Farbberechnung
AlphaArg0-2: Der Wert für die Operatoren 0, 1 und 2 für die Alphaberechnung
Blending RGB0-2: Die Farbquelle für die RGB-Werte und die Operatoren 0, 1 und 2
Blending Alpha0-2: Die Farbquelle für die Alpha-Werte und die Operatoren 0, 1 und 2
RepeatU/RepeatV: Gibt an, wie die Textur auf Wiederholungen reagiert (Koordinaten größer 1.0 und/oder kleiner 0.0)

Buttonfeld:
Die Buttons dienen dazu, die Materialien zu verwalten:

Delete: Löscht das aktuell selektierte Material
Create new: Erstellt ein neues Material mit einem bestimmten Namen
Load: Lädt ein Material aus einer Datei (*.bsmaterial)
Save: Speichert das selektierte Material in eine Datei (*.bsmaterial)

7.2 Texturkoordinateneditor
Menü -> Material -> Texture coordinate editor

Der Texturkoordinateneditor dient dazu, die Texturcoordinaten eines Models komfortabel zu bearbeiten.
Texturkoordinateneditor
Der Editor besteht aus 4 Teilen:
Mode: Bearbeitungsmodi
Object: Einstellungen über aktuell bearbeitendes Model
View: Einstellungen über die Darstellung
Copy: Kopiert Texturkoordinaten

Mode:
Hier wird der Bearbeitungsmodus definiert:

Select: Selektierungsmodus. Per Maus kann ein Selektionsrechteck gezogen werden, alle Punkte in diesem Rechteck werden selektiert
Move: Bewegungsmodus. Per Maus können die selektierten Punkte bewegt werden
Rotate: Rotierungsmodus. Per Maus können sie selekterten Punkte um deren Zentrum rotiert werden
Scale: Skalierungsmodus. Per Maus können sie selektierten Punkte von ihrem Zentrum aus skaliert werden
Open material editor: Der Materialeditor kann hierüber geöffnet werden


Object:
Das Modell und die Texturkoordinatenschicht, die bearbeitet werden sollen, können hier gewählt werden:

Mesh: Das Modell das bearbeitet werden soll
Texture stage: Die Texturkoordinatenschicht die bearbeitet werden soll

View:
Stellt die Darstellung ein:

Show grid: Aktiviert/Deaktiviert die Darstellung des Gitternetzes
Show replication: Aktiviert/Deaktiviert die Darstellung von Texturwiederholungen.
Reset view: Setzt die Kameraeinstellungen zurück

Copy:
Kopiert eine Texturkoordinatenschicht:

From: Gibt die zu kopierende Texturkoordinatenschicht an
To: Gibt an, wohin die Texturkoordinaten kopiert werden sollen
Copy: Kopiert die gewählte Texturkoordinatenschicht

7.3 Texturverschiebungen
Menü -> Material -> Edit texture offsets/repeatings

Mithilfe diesem Dialogs kann eine Textur auf einem Mesh oder Polygon einerseits verschoben werden und andererseits die Anzahl der Wiederholungen geändert werden.

Texturverschiebung
Texture offset U: Verschiebung der gebundenen Textur in horizontaler Richtung
Texture offset V: Verschiebung der gebundenen Textur in senkrechter Richtung
Texture repeat U: Anzahl der Wiederholungen der gebundenen Textur in horizontaler Richtung
Texture repeat V: Anzahl der Wiederholungen der gebundenen Textur in senkrechter Richtung
OK: Einstellungen werden übernommen
Cancel: Einstellungen werden verworfen
Änderung der Offseteigenschaften der Textur: U-Wert
auf 5.1 und V-Wert auf -6.4
Ergebnisbild der Offsetänderung

Änderung der Repeateigenschaften der Textur: U-Wert
auf 5 und V-Wert auf 2
Ergebnisbild der Repeatänderung
