3. Fenster/Ansichten


3.1 Bedienfeld


Hauptfenster

Das Hauptfenster besteht aus einer Toolbar (hier links), einer Menübar (oben), einer Statusbar (unten) und 4 Ansichten. Es gibt 3 orthogonale Ansichten: Draufsicht, Seitenansicht und Frontalansicht und eine frei drehbare perspektivische Ansicht.

In der Toolbar werden neben den Buttons für das Selektieren, Verschieben, Rotieren und Skalieren auch Buttons für Modifikatoren und das Einfügen von Objekten aufgeführt. Bitte beachten Sie, dass faste alle Generatoren und Modifikatoren nur als Plugins verfügbar sind. Wie sie Plugins installieren und laden erfahren Sie in Kapitel 10.1 Pluginmanager

Die Buttons sind der Übersichtlichkeit halber in verschiedenen Boxen gruppiert. Von diesen können Sie, um beispielsweise den Platz in der Toolbar besser auszunutzen, den Inhalt mit der enstprechenden Checkbox verstecken.


3.1.1 Objektexplorer
Objektexplorer

Mithilfe des Objektexplorers ist es möglich einzelne Meshes der Szene auszuwählen und umzubennen.

Um ein Mesh umzubennen klicken sie entweder doppelt auf den Menüeintrag oder Rechtsklicken sie auf den Menüeintrag und wählen sie die Option "Rename" aus.

Sie können mithilfe der "Strg" oder "Shift" Tasten mehrere Meshes gleichzeitig auswählen: Mithilfe der Strg-Taste wählen sie einzelne Einträge aus und wieder ab. Mithilfe der Shift-Taste können sie einen Bereich selektieren. Sie können auch die linke Maustaste gedrückt halten und sie so über den entsprechenden Bereich ziehen um alle Einträge in diesem Bereich auszuwählen.


3.1.2 Materialexplorer
Materialexplorer

Mithilfe des Materialexplorers ist es möglich Materialien zu verwalten. Sie können diese mithilfe des Kontextmenüs löschen (Menüpunkt "Delete"), umbennen(Menüpunkt "Rename") oder auf Objekte anwenden (Menüpunkt Apply). Falls das "Material-Edtior" Plugin installiert ist können Sie disen per Menüpunkt "Material" aufrufen.

Falls sie ein Objekt selektieren das ein Material besitzt wird der enstsprechende Eintrag im Materialexplorer blau angezeigt.


3.2 Ansichten
Textured + Normal

Die Szene lässt sich auf 2 Arten darstellen: 2D (Orthogonal) und 3D (Perspektivisch). Jede dieser Arten hat ihre Vor- und Nachteile, die insbesondere die Übersichtlichkeit und Funktionalität betreffen. Über das Kontextmenü (erreichbar über die rechte Maustaste) lässt sich jedes Fenster individuell einstellen.

Das Gitter:
In den Ansichten gibt es 2 verschiedene Arten von Gittern – das Minor-Gitter und das Major-Gitter. Das Major-Gitter dient dazu, volle Dekaden darzustellen und deswegen haben diese daher immer eine Entfernung von 10^n voneinander (z.B. 0.01, 1 oder 10, je nach Entfernung der Kamera). Eine weitere Unterteilung einer solchen Dekade nimmt das Minor-Gitter vor. Um beispielsweise ein Gitter zu erhalten, das alle 0.2 Einheiten eine Linie hat, wählt man eine Partition des Minor-Gitters von 5. Die Einstellungen des Gitters befinden sich in den Renderer-Einstellungen.

Kontextmenü
Grid: Aktiviert/Deaktiviert das Gitternetz
Maximized: Maximiert das aktuelle Ansichtsfenster
Normals: Aktiviert/Deakviert das Darstellen der Normalenvektoren
Textured: Darstellungsmodus: Das Face wird normal dargestellt. Außerdem wird das daran gebundene Material inklusive aller Texturen verwendet
Wireframe: Darstellungsmodus: Es wird das Drahtgittermodell dargestellt, das nur die Kanten, nicht aber das 'Innere' der Faces dargestellt
WireframeOverlay: Darstellungsmodus: Kombiniert Textured- und Wireframe-Modus. Das texturierte Modell enthält ein darüber gerendertes Drahtgittermodell
Points only: Darstellungsmodus: Es werden nur die Eckpunkte der Faces dargestellt
Solid: Darstellungsmodus: Das Face wird normal dargestellt, außerdem wird das daran gebundene Material verwendet (ohne Texturen!)


3.2.1 Orthogonale Ansicht (2D Ansichten)
Die orthogonalen Ansichtsfenster zeigen die Szene von allen Seiten – Vorne, Hinten, Oben, Unten, Rechts, Links. Die einzelnen Fenster zeigen jeweils eine Sicht auf die Szene parallel zu einer einzelnen Achse. In allen orthogonalen Ansichten lässt sich die Szene bearbeiten (Selektieren, Rotieren, ...).

Für die Darstellung stehen die oben erwähnten 5 Modi bereit:
  • Point
  • Wireframe
  • Solid
  • Textured
  • Wireframe Overlay

Die 3D Ansicht stellt die Szene in einer 3-dimensionalen Perspektive dar. In dieser Ansichten lässt sich die Szene nicht bearbeiten (Selektieren, Rotieren, ...). Sie dient 'nur' der Ansicht.

3.3 Screenshots


Screenshot dialog

Im Menüpunkt File->Take Screenshot findet sich dieser Dialog welcher es Ihnen erlaubt eine der 4 Ansichten in ein Bild umzuwandeln.

Die unterstützten Formate können gegebenfalls von ihrem System abhängen. Auf jedenfall werden jedoch das PNG (Portable Network Graphics) und das JPEG - Format unterstützt. Da es sich bei den Szenen um Computergraphiken handelt, empfielt sich das Speichern im PNG format, da bei JPEG-Blidern unansehliche "Artefakte" auftreten können.

Mithilfe des Drop-Down Menüs an der Oberseite des Dialogs können sie die gewüschte Ansicht auswählen. Sie können die Ansicht wie gewöhnlich drehen und zoomen.

In der oberen Rechten Ecke finden sie die Einstellungen für das Format in das das Bild gespeichert wird. Darunter können sie die Qualität des Bildes festlegen, falls sie das bild als JPEG speichern möchten.

Darunter können sie festlegen in welcher Auflösung das Bild gespeichert werden soll. Falls sie eine benutzerdefinierte Auflösung eintragen möchten, können Sie das tun, indem Sie die Checkbox "Costum" anklicken und die Werte in die entsprechenden Felder eintragen. Sie können diese Auflösung auch dauerhaft speichern, in dem Sie das "+"-Symbol anklicken. Mit dem "-"-Symbol löschen sie diese wieder.

An der Unterseite des Dialogs können Sie den Pfad angeben, in dem das Bild gespeichert werden soll.
