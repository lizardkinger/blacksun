5. Szenensteuerung

5.1 Selektieren

Aktionsleiste: Selektieren

Um mit 3D-Modellen einer Szene arbeiten zu können, müssen diese zuvor selektiert werden. Alle Modifikatoren arbeiten ausschließlich auf der Menge der selektierten Objekte. Aus diesem Grund gibt es bei der Auswahl von Objekten sehr große Freiheitsgrade

5.1.1 Selektion per Auswahl

Grundsätzliches Bei dieser Selektionsmethode können Szenenobjekte in den 2D-Objekten per Auswahlrahmen selektiert werden. Je nach Selektionsmodus kommen bei der Selektion andere Ergebnismengen zustande.

Parameter
Da Szenenobjekte je nach Sichtweise aus verschiedenen Bestandteilen bestehen, muß dem Modeller stets mitgeteilt werden, welche Sichtweise momentan benutzt werden soll.

Auswahl des momentanen Selektionsmodus:
Dies geschieht über die Selektionsleiste, welche 3 Sichtweisen anbietet.

Aktionsleiste: Modus Mesh
Mesh: Bei diesem Selektionsmodus werden Objekte nur als Ganzes betrachtet
Aktionsleiste: Modus Face
Polygon: Bei diesem Selektionsmodus werden die Polygone von Objekten betrachtet sowie auch Polygone die keinem Mesh zugeordnet sind.
Aktionsleiste: Modus Vertex
Vertex: Bei diesem Selektionsmodus werden die Vertices von Objekten betrachtet sowie auch Vertices die keinem Polygon zugeordnet sind.

5.1.2 Selektion per Menü

Menü -> Selection -> Select *

Für erweiterte Selektionsfunktionalität stehen im Menü weitere vorgefertigte Auswahloptionen bereit.

Select all Alle Szenenobjekte vom derzeitig eingestellten Selektionsmodus werden selektiert
Select nothing Alle Selektionen werden aufgehoben
Select invert Alle selektierten Objekte werden deselektiert, alle nicht selektierten dagegen schon selektiert
Select intersecting Alle Szenenobjekte die sich mit dem derzeit selektierten Rahmen schneiden, werden selektiert

5.2 Verschieben

Toolbox -> Action mode -> Move

Aktionsleiste: Verschieben
Das Verschieben bezieht sich auf die derzeitige Selektion. Der Benutzer muß dazu in einer 2D-Ansicht die Maustaste an einer Stelle gedrückt halten und in die gewünschte Richtung ziehen. Dann an der Endposition die Maustaste wieder loslassen.

Anmerkung
Je nach gewähltem 2D-Fenster wird aus der Mausbewegung eines Verschiebung der aktuellen Selektion errechnet.
So ergibt eine Verschieberichtung in positiver Y-Richtung in der Top-Ansicht eine tatsächliche Verschiebung in der negativen Z-Achse der Szene.

5.3 Rotieren

Toolbox -> Action mode -> Rotate

Aktionsleiste: Rotieren

Das Rotieren bezieht sich auf die derzeitige Selektion. Als Zentrum der Rotation wird der Mittelpunkt kompletten Selektion benutzt. Eine Rotation findet dann um diesen Mittelpunkt statt.

Um eine Rotation durchzuführen, muß der Benutzer in einem 2D-Fenster die Maustaste gedrückt halten und dann die Maus entweder um einen gewissen Abstand nach links oder nach rechts ziehen. Eine Bewegung nach links entspricht einer Rotation gegen den Uhrzeigersinn, eine Bewegung nach rechts einer Rotation im Uhrzeigersinn.
Während der Mausbewegung erscheint ein Rotationskreis zur visuellen Unterstützung des Benutzers.

Anmerkung
Je nach gewähltem 2D-Fenster wird aus der Mausbewegung eines Rotation der aktuellen Selektion errechnet.
So ergibt eine Rotation im Uhrzeigersinn in der Top-Ansicht ein anderes Ergebnis als beispielsweise in der Left-Ansicht.

5.4 Skalieren

Toolbox -> Action mode -> Scale

Aktionsleiste: Skalieren

Das Skalieren bezieht sich auf die derzeitige Selektion und verformt alle Objekte unabhängig voneinander.

Um eine Skalierung durchzuführen, muß der Benutzer in einem 2D-Fenster die Maustaste gedrückt halten und dann die Maus entweder um einen gewissen Abstand nach links oder nach rechts ziehen. Eine Bewegung nach links entspricht einer Verkleinerung, eine Bewegung nach rechts einer Vergrößerung.

Anmerkung
Je nach gewähltem 2D-Fenster wird aus der Mausbewegung eines Rotation der aktuellen Selektion errechnet.
So ergibt eine Rotation im Uhrzeigersinn in der Top-Ansicht ein anderes Ergebnis als beispielsweise in der Left-Ansicht.

5.5 Objekte verstecken

Menü -> Selection -> Hide *

Bei einer großen Anzahl von Szenenobjekten kann gegebenfalls die Übersicht leiden, daher gibt es die Möglichkeit Modelle zu verstecken und somit vom Renderprozess auszuschließen.

Hide selection Alle selektierten Objekte werden versteckt
Hide all Alle Objekte werden versteckt
Hide invert Alle versteckten Objekte werden sichtbar, wogegen sichtbare Objekte als versteckt markiert werden
Hide nothing Alle Objekte werden sichtbar

5.6 Objekte sperren

Menü -> Selection -> Freeze *

Die blau umrandeten Objekte stellen gesperrte Objekte dar
Falls man Teile einer Szene bereits fertig modelliert hat und nun ausschließlich nur noch mit einer bestimmten Auswahl arbeiten möchte, bietet sich neben dem Verstecken auch das Sperren von Objekten an. Dabei bleiben diese zwar sichtbar aber können nicht mehr selektiert werden.

Freeze selection Alle selektierten Objekte werden gesperrt
Freeze all Alle Objekte werden gesperrt
Freeze invert Alle gesperrten Objekte werden entsperrt, wogegen freie Objekte als gesperrt markiert werden
Freeze nothing Alle Objekte werden entsperrt

5.7 Zoomen

Menü -> Selection -> Zoom *

Um sich während einer Modellierungstätigkeit Übersicht zu verschaffen, gibt es die Möglichkeit auf bestimmte Regionen der Szene zu zoomen

Zoom to selection Die aktuelle Selektionsauswahl wird angezoomt
Zoom to all Alle Szenenobjekte werden sichtbar

5.8 Echtzeitspiegelung

Menü -> View -> Realtime mirror

Beispiel für Echtzeitspiegelung an der Y-Achse

Um die Modellierung von symmetrischen Objekten zu erleichtern, gibt es komfortable Möglichkeit des Spiegelns, die Echtzeitspiegelung. Es spiegelt die komplette Szene während der gesamten Bearbeitung an einer Ebene.

Tipps und Informationen:
Wichtig ist, dass die Szene nur zusätzlich gespiegelt dargestellt, allerdings keine neuen Vertexdaten generiert werden. Dies muss über das 'richtige' Spiegeln (Mirror) gemacht werden. Außerdem kann nur im Original selektiert werden. Sollte also eine Selektierung nicht möglich sein, liegt das daran, dass versucht wurde, im gespiegeltem Teil zu selektieren und nicht im Original.
