Nach dem Herunterladen des aktuellen Installers, wird die Installation mit einem Doppelklick (unter Unix meist mit einen einfachen Klick) auf das Paket starten.

Blacksun wurde erfolgreich installiert.

2.2 Deinstallation


Der Uninstaller wird unter
  • Unix durch Ausführen von "uninstall" im Installationsverzeichnis aufgerufen.
  • Windows entweder über den Startmenüeintrag oder über den Aufruf von "uninstall.exe" im Installationsverzeichnis aufgerufen.

Uninstaller nach erfolgreicher Deinstallation.

Die Einstellungen müssen, falls gewünscht, manuell gelöscht werden:
  • Unix: Das Verzeichnis <Home_Verzeichnis>/.config/FH-Landshut
  • Windows: Den Registrierungsschlüssel HKEY_CURRENT_USER\Software\FH-Landshut

2.3 Onlineupdate


Menü -> Help -> Check for new version

Startbildschirm des OnlineUpdater.

1. Schritt: Den Pfad zu der XML - Datei angeben, die die Informationen über neue Versionen enthält,
wobei im allgemeinen der Standartpfad ausreicht. Die Datei wird mit einem klick auf dem Start - Button ausgewertet.

2. Schritt: Auf der linken Seite (1) befindet sich eine Auflistung der bereits installierte Komponenten und ihrer Versionen. Auf der rechten Seite (2) finden sich die Komponenten, die zum Download stehen. Falls eine "Remote" neuer ist, als eine "Locale", kann diese durch den setzen des Hackens bei Update? (3) zum Download vorgemerkt werden. Mit einem klick auf Update (4) wird der Download gestartet.
