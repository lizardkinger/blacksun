1. Einführung

1.1 Allgemeines


Blacksun ist ein statischer 3D-Modeller, der von einem studentischen Projektteam an der FH-Landshut unter der Führung von Professor Dr. Hartmann, entworfen und erstellt wurde.

Screenshot
Im Vordergrund steht die Spieleentwicklung im Allgemeinen, welche stets auf einfach erstellbare 3D- Objekte angewiesen ist.
Diesen Modellen gemein sind die Eigenschaften, meist aus relativ wenigen Polygonen zu bestehen, mit einem Fokus auf schnell erzielbare Resultate.

Großer Wert wurde auf Plattformunabhängigkeit gelegt, was sich durch eine Unterstützung von Windows und Linux wiederspiegelt.
Der zweite große Entwicklungsschwerpunkt war die strikte Modularisierung der Projektkomponenten.

Ziel des Projektes waren neben dem reinen Softwareprodukt das Sammeln von Erfahrungen die ein derartiges Softwareprojekt von mittlerer Größe mit sich bringt.

1.2 Verwendete Techniken
  • Programmiersprache: C++
  • GUI-Bibliothek: Qt
  • Rendering-Bibliothek: OpenGL
